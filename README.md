**STUDENT PERFORMANCE TRACKING SYSTEM - Документация по функциям**

---

**ФАЙЛ: `main.go`**

**`package main`**
- **Назначение:** Объявление исполняемого пакета
- **Описание:** Только пакет с именем `main` может содержать функцию `main()` и компилироваться в исполняемый файл

**`import ( ... )`**
- **Назначение:** Импорт стандартных библиотек
- **Описание:**
  - `bufio` — буферизованное чтение ввода пользователя
  - `fmt` — форматированный ввод/вывод
  - `os` — работа с операционной системой (стандартный ввод/вывод, завершение программы)
  - `strconv` — преобразование строк в числовые типы
  - `strings` — операции со строками (обрезка, разделение, сравнение)

**`type Student struct { ... }`**
- **Назначение:** Определение структуры студента
- **Описание:**
  - `Name string` — полное имя студента (используется как уникальный идентификатор)
  - `Grades []float64` — список оценок студента

**`type Students map[string]Student`**
- **Назначение:** Псевдоним типа для карты студентов
- **Описание:** Отображение "имя → данные студента" для удобной работы и улучшения читаемости кода

---

**ФАЙЛ: `student.go`** *(методы структуры Student)*

**`func (s Student) average() float64`**
- **Назначение:** Вычисление среднего балла студента
- **Описание:**
  - Проверяет наличие оценок (возвращает 0.0 при пустом списке)
  - Суммирует все оценки
  - Возвращает среднее арифметическое (сумма ÷ количество)
  - **Формула:** `average = Σ(grades) / len(grades)`

---

**ФАЙЛ: `students_collection.go`** *(методы коллекции Students)*

**`func (ss *Students) addStudent(name string, grades []float64)`**
- **Назначение:** Добавление нового студента или обновление существующего
- **Описание:**
  - Принимает имя студента и список оценок
  - Создаёт или перезаписывает запись студента в карте
  - Использует указатель-приёмник для модификации исходной коллекции

**`func (ss Students) printAll()`**
- **Назначение:** Вывод информации о всех студентах
- **Описание:**
  - Проверяет, есть ли студенты в коллекции
  - Выводит каждого студента в формате:
    `ФИО: <имя> | Оценки: [оценки] | Средний балл: <среднее>`
  - Использует значение-приёмник (только чтение данных)

**`func (ss Students) filterByAverage(threshold float64) []Student`**
- **Назначение:** Фильтрация студентов по среднему баллу
- **Описание:**
  - Принимает пороговое значение среднего балла
  - Возвращает срез студентов, у которых средний балл строго меньше порога
  - **Критерий отбора:** `student.average() < threshold`

---

**ФАЙЛ: `input_processing.go`**

**`func readGrades(input string) ([]float64, error)`**
- **Назначение:** Парсинг строки с оценками
- **Описание:**
  - Принимает строку с оценками, разделёнными пробелами
  - Разбивает строку на отдельные элементы
  - Преобразует каждый элемент в число типа `float64`
  - Проверяет корректность:
    - Непустой ввод
    - Все элементы — числа
    - Оценки в диапазоне от 2 до 5
  - Возвращает срез оценок или ошибку в случае некорректных данных

**ФАЙЛ: `ui_controller.go`**

**`func main()`**
- **Назначение:** Главная функция программы, управляющая интерфейсом
- **Описание:**
  - Инициализирует сканер для чтения пользовательского ввода
  - Создаёт пустую коллекцию студентов
  - Выводит приветственное сообщение с доступными командами
  - Запускает бесконечный цикл обработки команд
  - Обрабатывает команды:
    - `add` — добавление студента
    - `list` — вывод всех студентов
    - `filter` — фильтрация по среднему баллу
    - `exit` — завершение программы

**Обработка команды `add`:**
- Запрашивает ФИО студента (не может быть пустым)
- Запрашивает оценки (через пробел)
- Парсит оценки с помощью `readGrades()`
- Добавляет студента через `addStudent()`
- Вычисляет и показывает средний балл

**Обработка команды `list`:**
- Вызывает метод `printAll()` для вывода всей коллекции

**Обработка команды `filter`:**
- Запрашивает пороговое значение среднего балла
- Проверяет корректность введённого числа
- Вызывает метод `filterByAverage()`
- Выводит отфильтрованных студентов или сообщение об отсутствии результатов

**Обработка команды `exit`:**
- Выводит прощальное сообщение
- Корректно завершает программу с кодом 0

**Обработка неизвестных команд:**
- Выводит сообщение о некорректной команде
- Показывает список доступных команд
